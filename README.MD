# Job Application Tracker

A simple web application to track job applications and their current status.

## Features
- Add job applications with company name, role, and status
- Track application status (Applied, Interview, Offer, Rejected)
- Delete applications
- Clean and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## What I Learned
- DOM manipulation using JavaScript
- Handling form submissions
- Dynamic UI updates
- Basic state management in the browser

## How to Run
1. Clone the repository
2. Open `index.html` in your browser

## Future Improvements
- Edit job entries
- Local storage support
- Filtering by status
